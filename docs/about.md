# 关于

这是我的个人知识库，用 [VitePress](https://vitepress.dev/) 搭建。

## 为什么做这个

学习过程中的笔记散落在各处——Notion、Typora、草稿纸...找个东西经常翻半天。索性整理成一个文档站，：

- 📂 **结构清晰** — 分类导航，一目了然
- 🔍 **快速搜索** — 几秒钟定位到需要的内容
- 🌐 **随时随地** — 部署到公网，手机电脑都能看
- 📝 **容易维护** — 写 Markdown 就行，自动生成页面

## 技术栈

- **框架**：[VitePress](https://vitepress.dev/)（Vite + Vue 驱动的静态站点生成器）
- **部署**：Vercel / GitHub Pages
- **域名**：待定
- **源码**：[GitHub](https://github.com/)
