# 计算机网络

## 网络分层模型

### OSI 七层模型

| 层次 | 名称 | 功能 | 协议/设备 |
|------|------|------|-----------|
| 7 | 应用层 | 为应用程序提供网络服务 | HTTP, DNS, SMTP |
| 6 | 表示层 | 数据格式转换、加密 | SSL/TLS |
| 5 | 会话层 | 建立/管理/终止会话 | RPC |
| 4 | 传输层 | 端到端可靠传输 | TCP, UDP |
| 3 | 网络层 | 路由寻址 | IP, 路由器 |
| 2 | 数据链路层 | 帧传输、差错控制 | MAC, 交换机 |
| 1 | 物理层 | 比特流传输 | 网线, 光纤 |

### TCP/IP 四层模型

1. **应用层** — HTTP, DNS, SMTP
2. **传输层** — TCP, UDP
3. **网际层** — IP, ICMP
4. **网络接口层** — Ethernet, Wi-Fi

## TCP 三次握手

```
客户端                    服务器
  | ---- SYN -------->    |
  | <-- SYN+ACK -----     |
  | ---- ACK -------->    |
```

## TCP 四次挥手

```
客户端                    服务器
  | ---- FIN -------->    |
  | <-- ACK ---------     |
  | <-- FIN ---------     |
  | ---- ACK -------->    |
```

## 常见面试题

- **TCP 为什么是三次握手而不是两次？** — 防止旧连接请求到达服务器导致错误建立连接
- **TCP 和 UDP 的区别？** — TCP 面向连接、可靠、有序；UDP 无连接、尽最大努力交付
