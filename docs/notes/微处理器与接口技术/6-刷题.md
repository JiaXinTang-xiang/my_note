## 二、计算题

**2**、假设图一中 D1~D8等 LED的正向压降 VF为 2V，LED点亮所需电流 IF为5mA，请问 R1~R8的作用是什么？试计算 R1~R8的值。

作用为：**==限流保护，防止过流烧毁 LED==**

值为：**==600Ω==**

计算过程如下：

$$R = \frac{V_{CC} - V_F}{I_F} = \frac{5V - 2V}{5mA} = \frac{3}{0.005} = 600\Omega$$

**3**、根据图三，写出光敏管电压V1与D的关系公式。当D分别为128与192时，V1分别是多少？

$$V_1 = V_{REF} \times \frac{D}{256} = 5 \times \frac{D}{256}$$

- D = 128 时：$V_1 = 5 \times \frac{128}{256} =$ **==2.5V==**
- D = 192 时：$V_1 = 5 \times \frac{192}{256} =$ **==3.75V==**

## 四、程序设计题

**7**、当按下S1时全灯 0.5s 闪烁，按下S2时奇偶组交替 1s 闪烁。

```c
#include <reg51.h>
sbit S1 = P1^0;
sbit S2 = P1^1;

void delay_ms(uint ms) {
    uchar i, j;
    for(; ms > 0; ms--)
        for(i = 0; i < 123; i++);
}

void main() {
    uchar mode = 0;
    while(1) {
        if(S1 == 0) mode = 1;
        if(S2 == 0) mode = 2;

        if(mode == 1) {          // S1: 全灯 0.5s 闪
            P2 = 0x00; delay_ms(500);
            P2 = 0xFF; delay_ms(500);
        } else if(mode == 2) {   // S2: 奇偶交替 1s
            P2 = 0xAA; delay_ms(1000);
            P2 = 0x55; delay_ms(1000);
        } else {
            P2 = 0xFF;
        }
    }
}
```

```c
#include <reg51.h>
uchar count = 0;
bit flag = 0;

void main() {
    TMOD = 0x01;
```

```c
#include <reg51.h>
uchar code seg[] = {0x3F,0x06,0x5B,0x4F,0x66,0x6D,0x7D,0x07,0x7F,0x6F};

    TH0 = (65536 - 50000) / 256;

}
```

**P2.0 输出 10ms PWM（50%占空比）**：

```c
#include <reg51.h>
sbit PWM = P2^0;

void main() {
    TMOD = 0x01;
    TH0 = (65536 - 5000) / 256;   // 5ms
    TL0 = (65536 - 5000) % 256;
    EA = 1; ET0 = 1; TR0 = 1;
    PWM = 1;
    while(1);
}

void T0_ISR() interrupt 1 {
    TH0 = (65536 - 5000) / 256;
    TL0 = (65536 - 5000) % 256;
    PWM = ~PWM;  // 每 5ms 翻转 → 周期 10ms, 占空比 50%
}
```
